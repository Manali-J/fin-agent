- Idea: A personal finance co-pilot that helps users understand money, create realistic plans, and make better financial decisions using their own financial data
- Core principles:
  - Context-aware: uses the user's financial profile
  - Task-focused: one job at a time
  - Action-oriented: not just insights, but decisions
- System structure:
  - Single user financial profile (state)
  - 3 core capabilities:
    - Explain
    - Plan
    - Analyze
  - Optional later: decision layer
