MVP 3 - Analyzer (Insights Engine)

Goal
- Identify patterns, risks, and meaningful insights from financial data

Input
- Structured data:
  - spending history
  - income data
  - budget vs actual
  - or manually provided numbers

Behavior
- No questioning flow (input is assumed complete)
- Focus on interpretation, not collection
- Compare:
  - categories
  - time periods
  - expected vs actual

Output
- Key patterns (e.g., rising spend)
- Risk signals (e.g., low savings rate)
- 2-3 insights that matter

Scope
- Trend detection
- Behavior analysis
- Financial health signals

Out of scope
- No budget creation
- No basic Q&A explanations
- No step-by-step planning

Success criteria
- Output answers:
  - "What's happening?"
  - "Why it matters?"
  - "What to watch?"
