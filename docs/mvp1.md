MVP 1 - Explainer (Q&A)

Goal
- Answer finance questions clearly and simply

Input
- User question (conceptual or practical)

Behavior
- No memory required (can optionally use context later)
- No multi-step flow
- No planning or analysis

Output
- Short, clear explanation
- Optional example if helpful

Scope
- Savings, investing, debt, taxes, basics
- Definitions + simple "when to use"

Out of scope
- No budgeting
- No personalization (initially)
- No follow-up workflows

Success criteria
- Answer is:
  - correct
  - simple
  - immediately useful
