MVP 2 - Planner (Budget Builder)

Goal
- Turn user's financial inputs into a practical monthly plan

Inputs (collected via conversation)
- Income (take-home)
- Fixed expenses (rent, EMI, bills)
- Variable spending (food, shopping, etc.)
- Savings goals
- Debt obligations (optional but important)

Behavior
- Multi-step flow (guided questions)
- Stores intermediate state
- Organizes data into categories

Output
- Monthly budget breakdown:
  - Needs
  - Wants
  - Savings
- Clear allocation
- 2-3 actionable suggestions

Scope
- Budgeting
- Expense categorization
- Basic prioritization

Out of scope
- No deep financial education
- No trend analysis
- No predictions

Success criteria
- User gets:
  - a complete budget
  - realistic allocations
  - clear next steps
